An AI Onboarding Experience for WordPress Sites. Discover features, plugins and site building packages with prompts.

== Description ==

Suggerence Onboarding is a WordPress plugin designed to enhance user experience by providing an AI-driven onboarding process. It helps users discover features, plugins, and site-building packages through intuitive prompts and recommendations.

The Plugin calls an external public API at https://api.ploogins.com/public/search and https://api.suggerence.com/v1/. The plugin sends the user query while searching for plugins to get recommended plugins using AI.
